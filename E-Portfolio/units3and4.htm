The Four Units in my Course

Web Design and Development

This unit covers all about HTML, or Hyper-Text Markup Language, CSS, or Cascade Style Sheets, and later on, Javascript. We also have access to Lynda.com which helps in the coding of this website you're on. Programming

This unit deals with coding in Processing/Java, understanding procedural and object-oriented programming in Processing. This unit is a lot of typing and a lot of thinking, because there is always problems that require solutions with coding.
